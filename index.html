💡 How the Sheets Fetch Process Works

1. Open Google Sheet
Set share: "Anyone with link can view"
➔
2. GViz Endpoint
Request tq?tqx=out:json for sheets
➔
3. JSONP Wrapper
Script callback executes in browser
➔
4. Direct Parse
Data populated in tables + localStorage
The standalone offline dashboard fetches directly from Google Sheets using the Google Visualization (GViz) data feed. Because this feed uses a JSONP response callback (e.g. _gvizCbMeta(data)), it completely bypasses Cross-Origin Resource Sharing (CORS) blocks, letting you fetch spreadsheets safely from a static local HTML file.
